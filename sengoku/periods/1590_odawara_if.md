# 🏯 1590年 小田原征伐（if展開）– 北条家の逆襲

## 📘 シナリオ背景

1590年、関白となった**豊臣秀吉**は、東国最大の独立勢力である**北条家**の討伐に乗り出す。  
20万の大軍を動員し、徳川家・上杉家・前田家・毛利家らを従えた**全国包囲網**が展開される。

だが、北条家にはまだ希望があった。  
徳川家康との縁、伊達政宗の独立志向、房総方面の水軍支援、西国の動揺……  
あなたの選択次第で、北条家は生き残るどころか、**東国の覇者**として歴史を塗り替えるかもしれない。
